# @tulilabs/nft-components

Tuli's NFT components allow you to easily create your own gallery or auction house with xNFT infrastructure.

This library provides the front-end display components to compliment `@tulilabs/nft-hooks`.

### Main components:

- [NFTPreview](?path=/docs/renderer-nftpreview--image)
  - Used to render a xNFT preview thumbnail
- [NFTFullPage](?path=/docs/renderer-nftfull--image)
  - Used to render a xNFT full page component
- [NFTDataProvider](?path=/story/renderer-nftdataprovider--page)
  - Used to compose a custom set of xNFT components
- [MediaConfiguration](?path=/story/renderer-mediaconfiguration--page)
  - Configure the text, theme, and network settings for the xNFT.
- [AuctionHouseList](?path=/docs/renderer-auctionhouselist--images)
  - Load a list of auctions for a given curator

### Key Features:

- [Server-side rendering](?path=/story/about-serverrendering--page)
- [Customizable theming / styling](?path=/story/theming-previewcomponent--preview-card)
- [Customizable information ordering / display](?path=/story/renderer-about--page)

### Quickstart

1. Install package:
```bash
yarn add @tulilabs/nft-components
```

2. Render a XNFT Thumbnail:

```tsx
import { NFTPreview } from "@tulilabs/nft-components";

export const Page = () => <NFTPreview id="3002" />;
```

3. Render a XNFT Full page:

```tsx
import { NFTFullPage } from "@tulilabs/nft-components";

export const Page = () => <NFTFullPage id="3002" />;
```
